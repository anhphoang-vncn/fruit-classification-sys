# Frcl service

### Python API for Fruit classification using Tensorflow

#### Setup service:

`$ pip install -r setup.txt`

#### Start service:

`$ python api_server.py`

#### ROUTE:

1. API test ==> [GET] /api/test
2. API fruit classifier ==> [POST] /api/classifier

#### Default PORT = 11500
